# Secure Email Setup Guide

## Overview
Instructions to configure secure email for end users.

## Setup Steps
1. Install the secure email client
2. Import your S/MIME certificate
3. Configure account settings:
   - Incoming: imap.example.com
   - Outgoing: smtp.example.com
4. Test sending and receiving encrypted emails

## Troubleshooting
- Ensure correct certificate is installed
- Verify network connectivity
