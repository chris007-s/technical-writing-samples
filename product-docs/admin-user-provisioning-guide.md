# Admin User Provisioning Guide

## Overview
This guide outlines the steps to create, manage, and remove admin users in the system.

## Steps
1. Login to the Admin Portal
2. Navigate to 'User Management'
3. Click 'Add User' and fill out required details
4. Assign appropriate roles and permissions
5. Save and verify the user account

## Notes
- Admin accounts must have MFA enabled.
- Review permissions quarterly.
